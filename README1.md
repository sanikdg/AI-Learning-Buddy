# AI Learning Buddy

AI Learning Buddy is an interactive educational web application built using Streamlit and Google Gemini API.

## Features

- 🤖 Custom AI Persona
- 📚 Topic Selection
- 💬 Ask AI
- 📖 Explain Topic
- 🌍 Real-Life Examples
- 📝 Quiz Generator
- 💡 Study Tips

## Technologies Used

- Python
- Streamlit
- Google Gemini API
- Google Colab
- ngrok

## How to Run

1. Install the required libraries:

```bash
pip install -r requirements.txt
```

2. Run the application:

```bash
streamlit run app.py
```
